ابتدا کد زیر را به هد اضافه کنید 
                                
                                    < link rel="stylesheet" href="fonts/fontawesome.min.css" >
.از هر کدام از فونتهای زیر که آیکن را برداشته اید، همان سی اس اس را به صفحه اضافه کنید .در این سی اس اس ها آدرس فونتی که همنام سی اس اس است داده شده است و در مرحله بعد بايد خود فونت هم در آدرس مناسب قرار داده شود .هر كدام را كه اضافه كرديد باز كنيد و آدرس فونت را مناسب با پروزه خود كنيد

< link rel="stylesheet" href="fonts/all.min.css" >
< link rel="stylesheet" href="fonts/solid.min.css" >
< link rel="stylesheet" href="fonts/brands.min.css" >
< link rel="stylesheet" href="fonts/fontawesome.min.css" >
< link rel="stylesheet" href="fonts/regular.min.css" >
< link rel="stylesheet" href="fonts/svg-with-js.min.css" >
< link rel="stylesheet" href="fonts/v4-font-face.min.css" >
< link rel="stylesheet" href="fonts/v4-shims.min.css" >
< link rel="stylesheet" href="fonts/v5-font-face.min.css" >

در داخل فلدر fonts فونتي كه استفاده شده است را اضافه كنيد. مثلا در اين پروزه از فونت solid استفاده شده است بنابراين فونت هاي fa-solid-900.ttf , woff به پروزه اضافه شده است و استايل solid.css نيز به پروزه افزوده شده
