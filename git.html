مدييت فايلها در گيت

بطور كلي مديريت فايلها و فلدرها در گيت سه فاز اساسي دارد:

working directory: يعني روي فايلهاي untrack كار كنيم. مثلا فقط git init كرده ايم و فايلها وارد شده اند.
staging area: يعني git add كن و شروع به track كن.
repository: يعني git commit كن و پروژه را در گيت شروع كن. اينجا فقط repository local داريم و بعدا ميتوانيم وارد repository remote مثل گيت هاب شويم.

ايجاد repository

براي ايجاد repository دو روش وجود دارد:

اول اينكه براي اولين بار ميخواهيم پروژه را در گيت شروع كينم >>> git init
با اين دستور فلدر git روي سيستم ايجاد ميشود.

دوم اينكه پروژه در گيت هاب وجود دارد و فقط يك كپي از آنرا ميخواهيم وارد سيستم خود كنيم >>> gti clone
به گيت هاب برويد و git clone را بزنيد و آدرس https را كپي كنيد . سپس به محيط cmd برويد و دستور git clone و در مقابل آن آدرس كپي شده را بزنيد.
