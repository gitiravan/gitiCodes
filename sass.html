ابتدا به آدرس زير برويد:
https://nodejs.org/en/download/current و node.js را نصب كنيد
مثل يك برنامه نصب ميشود
وارد vscode شويد و در قسمت ترمينال كد زير را تايپ كنيد تا مطمين شويد نصب به درستي انجام شده است:
node -v دقت كنيد ما بينش يه فاصله هست
حالا در قسمت پروژه خودمان بايد فلدر package.json را ايجاد كنيم.
براي اينكار بايد دستور زير را بنويسيم
npm init -y
-y يعني تمام موارد ظاهر شده را اينتر بزن تا رد شود
حالا در پروژه خودتان بايد فلدر pakage.json را مشاهده كنيد كه ساخته شده است
اگر دستور npm -v را بزنيم ميبينيم يك ورژن نشان ميدهد
حالا ميخواهيم sass را نصب كنيم مينويسيم: npm install -g node -sass
حالا pakage.json را باز ميكنيد و دستور زير را بجاي قسمت test مينويسيد: 
"sass": "node-sass -w scss/ -o dist/css --recursive"
يك فلدر dist بسازيد و داخل آن يك فلدر ديگر css بسازيد
يك فلدر scss بسازيد و داخل آن main.scss بسازيد
خود main.css هم به صفحه اضافه كنيد.
حالا با تايپ دستور npm run sass بايد دستور بالا اجرا شود و كدها اجرا و تبديل شوند
